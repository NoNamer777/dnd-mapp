# DmaTooltip

This directive allows you to add a plain tooltip, which follows the DnD Mapp style definition, to any element. With this
tooltip you can show some extra context about what the target element does.

<Canvas of={DmaTooltipStories.Common} />

## Usage

In order to use this directive, you first need to import its module like so:

```typescript
import { NgModule } from '@angular/core';
import { DmaTooltipModule } from '@dnd-mapp/shared-components';

@NgModule({
    imports: [DmaTooltipModule],
})
class MyModule {}
```

After that, you'll be able to use the directive in you component's template like so:

```html
<button dma-button="filled" [dmaTooltip]="tooltipText" [dmaTooltipPosition]="position">Button</button>
```
